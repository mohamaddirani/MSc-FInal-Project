39 The function expects the GPT model to respond with a JSON object in the format:

42 "destination": "point A"

44 Args:
45 text (str): The natural language command to parse.
46 Returns:
47 dict or None: A dictionary with keys "robot_id" and "destination" if parsing is successful,
48 or None if an error occurs.

50 """
51 prompt = f"""
52 You are an API that receives robot commands in natural language and returns JSON only.
53
54 Each command might include:

57
58 If no robot is mentioned, return null for "robot_id".
59
60 Always return in this JSON format:
61 {{"robot_id": "Robot1" or null, "destination": "point A"}}
62
63 Examples:
64 Input: "Robot1 go to point A"
65 Output: {{"robot_id": "Robot1", "destination": "point A"}}

67 Input: "Go to point B"
68 Output: {{"robot_id": null, "destination": "point B"}}

70 Input: "Send someone to point C"
71 Output: {{"robot_id": null, "destination": "point C"}}
72
73 Input: "Robot zero move to point C"
74 Output: {{"robot_id": "Robot0", "destination": "point C"}}
