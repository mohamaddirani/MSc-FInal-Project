15class PathExecutor:
16 """
17 Executes a planned path for a robot in a simulation environment, handling movement, obstacle avoidance, and goal alignment.
18 Attributes:
19 sim: The simulation environment object.
20 motion: An instance of RobotMotion for controlling robot movement.
21 robot: The robot object or identifier.
22 cleared (str): Flag indicating if an obstacle has been cleared ("True"/"False").
23 Failed (str): Flag indicating if path execution has failed ("True"/"False").
24 Methods:
25 __init__(sim, robot, wheels):
26 Initializes the PathExecutor with the simulation, robot, and wheels.
27 async get_position():
28 Retrieves the current (x, y) position of the robot.
29 async follow_path(path):
30 Follows a given path (list of waypoints), moving the robot to each goal while handling obstacles and alignment.
31 Returns "DONE" if the path is completed, "FAILED" if execution fails, or "replanned" if replanning is triggered.
32 async move_to_goal(goal):
33 Moves the robot towards a specified goal, checking for obstacles and alignment.
34 Returns "MOVING" if still in progress, "DONE" if the goal is reached, "FAILED" if blocked, or "replanned" if replanning is needed.
35 async check_alignment(direction, coordinates_for_goal):
36 Checks if the robot is aligned with the goal along the X or Y axis based on obstacle direction and goal coordinates.
37 Returns "X-AXIS ALIGNED", "Y-AXIS ALIGNED", or None.
38 async handle_obstacle(Robot, direction, goal, dx, dy):
39 Handles obstacle avoidance by moving towards a temporary goal to bypass the obstacle.
40 Updates the 'cleared' and 'Failed' flags based on the outcome.
41 """
